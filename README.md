# basketbot

Telegram-бот на `aiogram 3.x`, который отслеживает **неанонимные** опросы, пересоздаёт их **от имени бота** и публикует отдельное сообщение с автообновляемыми результатами голосования.

## Как это работает

1. Администратор вручную публикует **неанонимный** опрос в чате.
2. Бот видит этот опрос.
3. Бот создаёт **такой же опрос от своего имени**.
4. Бот пытается удалить исходный админский опрос.
5. Бот отправляет отдельное сообщение с результатами.
6. Бот обновляет счётчик двумя способами:
   - через `poll_answer`, когда знает, кто именно проголосовал;
   - через `poll`, когда Telegram присылает актуальное состояние самого опроса.

Такой двойной механизм нужен, чтобы не пропускать голоса, если отдельный `poll_answer` пришёл с задержкой или потерялся в процессе обработки.

## Что умеет бот

- отслеживает новые сообщения с опросами;
- игнорирует анонимные опросы;
- дублирует неанонимный опрос от имени бота;
- удаляет исходный админский пост с опросом;
- создаёт отдельное сообщение с результатами;
- обновляет результаты при каждом новом голосе;
- дополнительно синхронизирует счётчики по `poll`-обновлениям;
- отправляет отдельное сообщение при достижении порога по первому варианту ответа;
- показывает стоимость игры, рассчитанную по числу голосов в первом варианте;
- корректно обрабатывает смену голоса;
- поддерживает несколько опросов одновременно.

## Требования

- Python `3.10+`
- Telegram Bot Token от `@BotFather`
- права администратора у бота в чате

## Важные права бота

Чтобы схема работала, у бота в группе должны быть права:

- на отправку сообщений;
- на создание опросов;
- на удаление сообщений.

Если права на удаление сообщений нет, бот сможет создать свой опрос, но старый админский опрос может остаться в чате.

## Установка

### 1. Склонировать проект

```bash
git clone <URL_РЕПОЗИТОРИЯ>
cd basketbot
```

### 2. Создать виртуальное окружение

```bash
python -m venv .venv
source .venv/bin/activate
```

Для Windows:

```bash
.venv\Scripts\activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

## Настройка токена

Бот сначала ищет токен в переменной окружения `token`. Если она не задана, бот проверяет `BOT_TOKEN` для обратной совместимости и только потом использует константу `TOKEN` в файле `bot.py`.

### Вариант 1. Через переменную окружения

Linux / macOS:

```bash
export token="123456:ABCDEF"
python bot.py
```

Windows PowerShell:

```powershell
$env:token="123456:ABCDEF"
python bot.py
```

### Вариант 2. Через `bot.py`

Откройте файл `bot.py` и вставьте токен вашего бота в константу `TOKEN`:

```python
TOKEN = "PASTE_YOUR_BOT_TOKEN_HERE"
```

## Railway

Если вы деплоите проект на Railway, добавьте переменную окружения:

- `token` = токен, который вы получили у `@BotFather`

Шаги в Railway:

1. Откройте ваш сервис.
2. Перейдите в раздел **Variables**.
3. Создайте переменную `token`.
4. Вставьте реальный токен бота без кавычек и лишних пробелов.
5. Перезапустите сервис.

## Как создать бота в Telegram

### 1. Откройте `@BotFather`

В Telegram найдите официального бота `@BotFather`.

### 2. Создайте нового бота

Отправьте команду:

```text
/newbot
```

Дальше BotFather попросит:

- указать имя бота;
- указать username бота, который должен оканчиваться на `bot`.

После этого BotFather отправит вам токен.

### 3. Отключите Privacy Mode

Чтобы бот видел сообщения с опросами, отправьте в `@BotFather` команду:

```text
/setprivacy
```

Дальше:

1. Выберите вашего бота.
2. Выберите `Disable`.

Без этого бот в группе может не получать обычные сообщения и не увидит новое сообщение с опросом.

## Как добавить бота в Telegram-чат

### В группу или супергруппу

1. Откройте нужную группу.
2. Нажмите **Добавить участника**.
3. Найдите вашего бота по username.
4. Добавьте его в группу.

### Выдать права админа

1. Откройте информацию о группе.
2. Перейдите в список администраторов.
3. Назначьте бота администратором.
4. Выдайте права на:
   - отправку сообщений;
   - удаление сообщений;
   - управление опросами, если это требуется в вашем интерфейсе Telegram.

## Как запустить бота

Локально:

```bash
python bot.py
```

На Railway приложение запускается обычной командой старта проекта, главное — чтобы была задана переменная `token`.

## Как пользоваться

1. Добавьте бота в группу.
2. Убедитесь, что у него выключен Privacy Mode.
3. Убедитесь, что у него есть права администратора.
4. Создайте **неанонимный** опрос вручную через стандартный интерфейс Telegram.
5. Бот удалит исходный опрос, создаст такой же опрос от своего имени и отправит отдельное сообщение с результатами.
6. Когда пользователи голосуют или меняют голос в новом опросе от бота, сообщение с результатами будет автоматически обновляться.
7. Если в первом варианте ответа становится больше 7 голосов, бот один раз отправляет сообщение:

```text
Достигнуто: 7 игроков
Разблокирован новый персонаж — @Ilhomchik_R
```

8. Внизу сообщения с результатами бот показывает строку `Стоимость игры: xxx руб`, где `xxx` рассчитывается по формуле `2250 / количество голосов в первом варианте`.

## Важные замечания

- Бот работает только с **неанонимными** опросами.
- Автообновление работает для опроса, который бот пересоздал **от своего имени**.
- Дополнительная синхронизация по `poll` помогает подтягивать актуальные числа голосов, даже если часть `poll_answer` пришла неидеально.
- Уведомление о разблокировке персонажа отправляется один раз для каждого опроса после превышения порога в 7 голосов у первого варианта.
- Стоимость игры рассчитывается по первому варианту ответа. Если голосов ещё нет, вместо числа выводится `—`.
- Все данные хранятся в памяти процесса. После перезапуска бота текущие результаты будут сброшены.
