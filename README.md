# basketbot

Telegram-бот на `aiogram 3.x`, который отслеживает **неанонимные** опросы, созданные вручную через стандартный `Telegram Poll`, и публикует отдельное сообщение с живыми результатами голосования.

## Что умеет бот

- отслеживает новые сообщения с опросами;
- игнорирует анонимные опросы;
- сохраняет `poll_id` и создаёт отдельное сообщение с результатами;
- обновляет результаты при каждом новом голосе;
- корректно обрабатывает смену голоса;
- поддерживает несколько опросов одновременно.

## Требования

- Python `3.10+`
- Telegram Bot Token от `@BotFather`

## Установка

### 1. Склонировать проект

```bash
git clone <URL_РЕПОЗИТОРИЯ>
cd basketbot
```

### 2. Создать виртуальное окружение

```bash
python -m venv .venv
source .venv/bin/activate
```

Для Windows:

```bash
.venv\Scripts\activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

## Настройка токена

Бот сначала ищет токен в переменной окружения `BOT_TOKEN`. Если переменная не задана, используется константа `TOKEN` в файле `bot.py`.

### Вариант 1. Через переменную окружения

Linux / macOS:

```bash
export BOT_TOKEN="123456:ABCDEF"
python bot.py
```

Windows PowerShell:

```powershell
$env:BOT_TOKEN="123456:ABCDEF"
python bot.py
```

### Вариант 2. Через `bot.py`

Откройте файл `bot.py` и вставьте токен вашего бота в константу `TOKEN`:

```python
TOKEN = "PASTE_YOUR_BOT_TOKEN_HERE"
```

## Railway

Если вы деплоите проект на Railway, добавьте переменную окружения:

- `BOT_TOKEN` = токен, который вы получили у `@BotFather`

Шаги в Railway:

1. Откройте ваш сервис.
2. Перейдите в раздел **Variables**.
3. Создайте переменную `BOT_TOKEN`.
4. Вставьте реальный токен бота без кавычек и лишних пробелов.
5. Перезапустите сервис.

Если переменная не задана, бот завершится с понятной ошибкой, а не с падением из-за тестовой строки `PASTE_YOUR_BOT_TOKEN_HERE`.

## Как создать бота в Telegram

### 1. Откройте `@BotFather`

В Telegram найдите официального бота `@BotFather`.

### 2. Создайте нового бота

Отправьте команду:

```text
/newbot
```

Дальше BotFather попросит:

- указать имя бота;
- указать username бота, который должен оканчиваться на `bot`.

После этого BotFather отправит вам токен.

### 3. Отключите Privacy Mode

Чтобы бот видел сообщения с опросами, отправьте в `@BotFather` команду:

```text
/setprivacy
```

Дальше:

1. Выберите вашего бота.
2. Выберите `Disable`.

Без этого бот в группе может не получать обычные сообщения и не увидит новое сообщение с опросом.

## Как добавить бота в Telegram-чат

### В группу или супергруппу

1. Откройте нужную группу.
2. Нажмите **Добавить участника**.
3. Найдите вашего бота по username.
4. Добавьте его в группу.

### Выдать права админа

Чтобы бот стабильно видел сообщения с опросами и мог отправлять/редактировать сообщения, рекомендуется выдать ему права администратора:

1. Откройте информацию о группе.
2. Перейдите в список администраторов.
3. Назначьте бота администратором.
4. Дайте минимум права на:
   - отправку сообщений;
   - редактирование сообщений, если это требуется настройками чата.

## Как запустить бота

Локально:

```bash
python bot.py
```

На Railway приложение запускается обычной командой старта проекта, главное — чтобы была задана переменная `BOT_TOKEN`.

## Как пользоваться

1. Добавьте бота в группу.
2. Убедитесь, что у него выключен Privacy Mode.
3. Убедитесь, что у него есть доступ к сообщениям в чате.
4. Создайте **неанонимный** опрос вручную через стандартный интерфейс Telegram.
5. После появления опроса бот отправит отдельное сообщение вида:

```text
📊 Результаты голосования

Вариант 1 — 0️⃣
Вариант 2 — 0️⃣
Вариант 3 — 0️⃣

Всего: 0 человек
```

6. Когда пользователи голосуют или меняют голос, бот автоматически обновляет это сообщение.

## Важные замечания

- Бот работает **только с неанонимными опросами**.
- Если опрос анонимный, бот его проигнорирует.
- Все данные хранятся в памяти процесса. После перезапуска бота текущие результаты будут сброшены.
